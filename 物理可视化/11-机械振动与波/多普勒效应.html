多普勒效应公式（声源运动）

f' = f × v / (v - vs)

其中：

  • f' 是观察者感知的频率
  • f 是声源的实际频率 (500 Hz)
  • v 是声速 (340 m/s)
  • vs 是声源速度（向右为正，向左为负）

关键点：

  • vs < 0（负值）：声源向左运动
  • vs = 0：声源静止
  • vs > 0（正值）：声源向右运动

多普勒效应总结：

声源运动 相对观察者 频率变化
向左 (v<0) 朝向观察者 升高
向左 (v<0) 远离观察者 降低
向右 (v>0) 朝向观察者 升高
向右 (v>0) 远离观察者 降低

声源速度双向控制的多普勒效应

本演示完整展示了声源向左或向右运动时的多普勒效应。速度滑块范围为-80到80 m/s，负值表示向左运动，正值表示向右运动，0表示静止。

速度方向与多普勒效应的关系：

  • 声源向左运动 (vs < 0)
    • 声源在观察者右侧：朝向观察者 → 波面压缩 → 频率升高
    • 声源在观察者左侧：远离观察者 → 波面拉伸 → 频率降低
  • 声源向右运动 (vs > 0)
    • 声源在观察者左侧：朝向观察者 → 波面压缩 → 频率升高
    • 声源在观察者右侧：远离观察者 → 波面拉伸 → 频率降低

本演示通过以下方式直观展示双向运动的多普勒效应：

  1. 双向速度滑块：范围-80到80 m/s，负值向左，正值向右
  2. 方向控制按钮：快速设置向左、停止、向右三种状态
  3. 波面颜色编码：红色表示压缩（频率升高），蓝色表示拉伸（频率降低）
  4. 详细的波面分析：显示观察者位置处波峰间距的变化

通过本演示，学生可以直观理解：

  • 速度的矢量性质（正负表示方向）
  • 多普勒效应的本质是相对运动，与绝对运动方向无关
  • 声源与观察者的相对位置决定频率变化方向
  • 多普勒效应公式中vs的正负号对结果的影响

这种设计特别适合教学，因为它完整展示了声源在任意方向运动时的多普勒效应，帮助学生全面理解这一重要物理现象。
