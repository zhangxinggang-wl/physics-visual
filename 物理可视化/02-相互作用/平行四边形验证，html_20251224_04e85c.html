平行四边形定则公式
F合 = √(F₁² + F₂² + 2F₁F₂cosθ)
θ = |θ₁ - θ₂| (两力之间的夹角)
合力方向：tanφ = (F₁sinθ₁ + F₂sinθ₂) / (F₁cosθ₁ + F₂cosθ₂)

平行四边形定则说明

平行四边形定则是矢量合成的基本法则，用于计算两个矢量（如力、速度、位移等）的合成效果。

1. 将两个矢量的起点放在一起，以它们为邻边作平行四边形。

2. 从共同起点出发的对角线即为这两个矢量的合成矢量（合力）。

3. 合力的大小不仅取决于两个分力的大小，还取决于它们之间的夹角。

4. 当两力方向相同时，合力最大（两力之和）；当两力方向相反时，合力最小（两力之差）。

5. 本演示中，红色箭头表示力F₁，蓝色箭头表示力F₂，绿色箭头表示合力。灰色虚线表示平行四边形。
